    Mobile App Design Project

    Design a bike safety quiz app to your specifications, provided as a final project. Read the instructions and specifications carefully. Design a mobile app and submit a mockup design with a description of the overall flow of the app. Submitted mobile app designs will be evaluated according to the scoring criteria in the table below.

    Instruction:
    1. Read the provided specification carefully
    2. Identify the components and interfaces in the specification
    3. Create a table of page components
    4. Draw the mockup using pencil and paper or other design tools
    5. Create and submit design documents using one of the following formats:d
      Option 1: Create a word document and describe the elements and interface of your app. Draw a mockup, insert it into this document and submit.
      Option 2: Open the design tool and draw the model. Add descriptions of elements and interfaces, save and submit as pdf, png or jpeg files.

    Specification:
    • Purpose of this app: Bicycle safety education for elementary school students
    • You will need at least 3 pages: a home page, a troubleshooting page, and a quiz results page.
    • The homepage should include the title of this app.
    • The problem-solving page should include the following parts::
      • Progress bar showing current question number and percent complete
      • Question
      • Direction buttons
      • Responsive function (tells correct or incorrect answer)
    • The quiz results page should contain the total score of the quiz and 2 menu buttons. Take the quiz again and go home.

    Notes:

    Please carefully read the followings.

    All graphics used in app design must be created or drawn by you. However, for cell-phone image, you can use royalty-free images or use templated provided. You receive 0 point if downloaded graphics are included.

    Your drawing skills are not evaluated, only whether the items are well placed in the right place according to the specifications are considered.

    The provided app design is an example. You don't need to follow that. You must create your own design not imitating this.
