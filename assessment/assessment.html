    Mobila App Design Project

    Design a bike safety quiz app according to the specifications provided as a final project. Read the instructions and specifications carefully. Design a mobile app and submit a mockup design with a description of the overall flow of the app. Submitted mobile app designs will be evaluated according to the scoring criteria in the table below.

    Instruction:
    1. Read the below specification
    2. Identify composing elements and interface from the specification
    3. Create a table for elements
    4. Draw the mock-up using a pencil-and-paper or any design tools
    5. Submit the design document for one of the following formats.
      Option 1: Create word document, describe the elements and interfact of the app. Draw the mock-up and embed it into this document. Submit it.
      Option 2: Open the design tool and draw the model. Add descriptions for elements and interfaces, and save and submit as pdf, png or jpeg files.

    Specification:
    • Purpose of this app: Bicycle safety education for elementary school students
    • At least three pages are required: home page, problem-sovling page, and quiz-result page
    • Home page must contain the title of this app
    • Problem-solving page must contain the following parts:
      • progress bar of showing the current question number and percentage of completion
      • question
      • direction buttons
      • reaction features (correct or wrong)
    • Quiz-result page must contain the totle score of the quiz and two menu buttons, quiz again and go home

    Notes:

    Please carefully read the followings.

    All graphics used in app design must be created or drawn by you. However, for cell-phone image, you can use royalty-free images or use templated provided. You receive 0 point if downloaded graphics are included.

    Your drawing skills will not be evaluated, only if the content that meets the specification is well placed on the appropriate places will be assessed.

    The provided app design is an example. You don't need to follow that. You must create your own design not imitating this.
