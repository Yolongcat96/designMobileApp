    Welcome to Creating my First Flutter App Class

    Do you want to create a mobile app for your teaching or do you want to create a variety of mobile apps for education? However, if you are having difficulty because you have never learned programming at all, this course will help you get started. This course consists of practicing basic knowledge of mobile app development, setting up a development environment, practicing simple Flutter programming, and finally, developing a simple couter app. This lesson will give you an overview of the process of developing a mobile app using Flutter.

    Learning Objectives

    After successfully completing this course, you will be able to:

    • Gain a clear understanding of the overall process of mobile app development.
    • Know how to convert an idea into a design.
    • Know how to download and install a development tool kit, Android Studio, and understand features of it.
    • Know a programmjing language, Flutter, and practice to write basic code.
    • Create one single view app, Counter App, and run it with a simulator or actual device.
