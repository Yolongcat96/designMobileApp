    Mobile App Review

    In this lesson, you will review Quizizz, a quiz mobile app. In this activity, you will practice identifying the components of a mobile app page.

    1. Identify big blocks from the page of an app.

    Create groups by merging items on the screen for the same purpose

    Page title

    User related feature

    List of quizzes

    Bottom tab

    2. Identify all items per block.

    Title Block

    Title block is composed of the followings:

    Company or app logo

    Hamberger menu: display all menus when clicking this icon

    User Information Block

    User Info block is composed of the followings:

    User name & user profile image

    Edit a user's information and view activities completed or participated by the user

    Progress bar of showing learning process

    Direction texts

    Quiz List Block

    Quiz List Block is composed of the followings:

    Main vertical slideview: contains multiple sub slideview blocks

    Sub horizontal slideview

    Section title and view all button

    Clickable button to move to the individual quiz main view

    Bottom Tab Block
    Bottom Tab Block is composed of the followings:

    Tab icon

    Tab name
